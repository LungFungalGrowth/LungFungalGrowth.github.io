Modeling iron handling by monocyte derived macrophages challenged by A. fumigatus

Iron is essential for all living organisms. A characteristic immune response to infections is the sequestration of iron by macrophages to keep it away from pathogens. We have created a generalized Boolean model of how monocyte derived activate an iron metabolism program after sensing A. fumigatus based on the transcriptomics data (see the Bioinformatics section) as well as the literature.

The code for the implementation of the intracellular model, as well as a more thorough description can be found here.
